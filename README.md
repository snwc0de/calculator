# CALCULATOR
This project involves the development of an on-screen calculator using JavaScript, HTML, and CSS.

# operator buttons
This project contains all of the basic math operators you typically find on calculators:
1. add (+)
2. subtract (-)
3. multiply (/)
4. divide (*)
5. equals (=)
